# Grove - 3Axis Digital Accelerometer

## 腳位接法
| HC05 pin | Mbed pin   |
| -------- | ---------- |
| 1 - SCL  | P27 - SCL  |
| 2 - SDA  | P28 - SDA  |
| 3 - Vcc  | Vout - 3.3V|
| 4 - GND  | GND        |
